# zebra

A Clojure wrapper for Stripe.

Inspired by clojurewerkz/elephant: https://github.com/clojurewerkz/elephant

## Install

Add the following to your `project.clj` file:

```clj
[io.logicblocks/zebra "0.2.0-RC0"]
```
